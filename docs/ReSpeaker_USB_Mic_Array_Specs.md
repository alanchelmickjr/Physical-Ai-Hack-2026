# Seeed ReSpeaker USB Mic Array, 4-Mic Far-Field

**Manufacturer:** Seeed Studio
**Part Number:** 107990193
**Price:** $69.00

## Overview

USB-powered 4-mic far-field array with XMOS XVF-3000 DSP, beamforming, noise suppression and 12 RGB LEDs for smart speakers, voice assistants and meeting rooms.

The Seeed Studio ReSpeaker USB Mic Array is a fully enclosed, plug-and-play 4-microphone far-field voice capture device powered by the XMOS XVF-3000 DSP. It delivers clear, processed audio using advanced beamforming, noise suppression, de-reverberation, acoustic echo cancellation, and voice activity detection. With USB Audio Class 1.0 support, 12 programmable RGB LEDs, and a built-in 3.5 mm audio output, it provides a ready-to-use voice interface for smart speakers, conferencing systems, robots, and a wide range of voice-enabled products.

## Key Differences from ReSpeaker Mic Array V2.0

- Optimized power circuit
- Audio jack and Micro USB port repositioned to the rear for enhanced usability
- Mold-injected enclosure that eliminates tooling costs and accelerates time-to-market
- Carefully engineered acoustic structure

## Features

- Plug-and-play USB device; compatible with Windows 10/8/7, macOS, Linux, and Android (OTG supported)
- Far-field voice pick-up up to 5 meters with 360° capture pattern
- Advanced on-chip acoustic algorithms:
  - DOA (Direction of Arrival)
  - AEC (Acoustic Echo Cancellation)
  - AGC (Automatic Gain Control)
  - NS (Noise Suppression)
- Four high-performance digital MEMS microphones
- 12 programmable RGB LED indicators for status and feedback
- Built-in 3.5 mm audio jack for headphones or active speakers
- USB Audio Class 1.0 for simple driverless operation
